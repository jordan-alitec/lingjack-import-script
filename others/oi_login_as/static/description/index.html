Log In as Another User

"The Smart Choice for System Administrators"

This module allows the administrator to log in as/impersonate a user.

Description

 As a system administrator, you usually need to log in as another user to do things like setting up profiles, ensuring permissions are set correctly, and completing other configuration tasks.


“Log In as Another User” allows administrators to have easy access to the system as a specific user and thus solve problems or provide better and faster support. This module enables the admins to log in as a particular user and handle any situation without wasting any time at all. This is a valuable solution for testing and verifying user access rights.

Also, this feature offers two options for logging in:

  → As a specific user.

  → On behalf of a user who belongs to a particular access group.

Menus

Key Features

  • Great discretion in the logging operation.

  • High flexibility in user selection and relevant access.

  • Easy switching between users.

  • Both single-user or access group login options.
